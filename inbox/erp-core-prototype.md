# Raw Requirement จากปอนด์ (2026-07-06)

อยากได้ระบบที่ช่วยบริหารจัดการ ให้ลูกค้า ที่เป็น cosmetic manufactory โดยหลักๆ เลยจะเป็นเรื่อง การจัดการข้อมูลลูกค้า เปิด Purchasing Order และมีการตรวจสอบ stock ซึ่งเป็นส่วนผสมของสูตรของเครื่องสำอาง หลังจากนั้นจะส่งงานไปให้ฝ่ายการผลิต มีการตรวจสอบ assign และดำเนินการผลิต มีการส่งการตรวจสอบสถานะ รวมถึงการจัดส่ง และ ออก invoice และการเรียกเก็บเงิน

ต้องมีการจัดการเรื่องของ User Management ที่จะเข้าใช้ระบบ ลง log ทุกครั้ง มี dashboard ของแต่ละส่วนงานต่างๆ ขอ prototype มาดูหน่อย
