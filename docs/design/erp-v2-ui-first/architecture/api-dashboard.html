API Spec — Dashboard + Drill-down (ครบ 29 tile)

แดชบอร์ด 7 แผนก. ทุก tile มี สูตรนับ + ชนิด (event/state) + summary endpoint + list(drill) endpoint + drill columns → deep link. สิทธิ์เห็น = Read ของ module ปลายทาง (ไม่ใช่ role) (ปอนด์ยืนยัน). date filter (preset/custom) มีผลทุก tile. auto-refresh 15s คง filter+view. อ้าง po-spec-depth-audit §2 + functional-spec/dashboard.

← Architecture Functional spec (BA) Mockup: dashboard

ในหน้านี้

สิทธิ์ = Read (ไม่ใช่ role) date filter + event/state รูปแบบ endpoint Sale (5) Stock (4) Production (4) QC (4) Shipping (4) Finance (4) Admin (4) error

สิทธิ์การเห็น = Read permission (ไม่ใช่ role)

ปอนด์ยืนยัน: การเห็นแผนกในแดชบอร์ดยึด สิทธิ์ Read ของ module นั้น ๆ ไม่ใช่ชื่อ role — role ไหนก็ตามที่ role_permission มี r=true ของ module = เห็นแผนกนั้น. หลาย Read = สลับได้หลายแผนก. สอดคล้องกับ noti (ADR-005/007) ที่ใช้ Read bit เหมือนกัน.
GET/api/dashboard/departments
ตัวสลับแผนก (multi-dept switcher): คืนรายชื่อแผนกที่ user มีสิทธิ์ Read (sale/stock/production/qc/shipping/finance/admin) — UI แสดงเป็น tab/selector. ไม่มี Read แผนกใด = ไม่โผล่ในตัวสลับ + เข้า URL ตรง → 403. อ่านจาก role_permission ของ user

date filter + นิยาม event vs state

date filter (query param ทุก endpoint ด้านล่าง): ?range=today|week|month|custom (default month) · ถ้า custom → from & to. มีผลกับ tile ชนิด event เท่านั้น. refresh/auto-refresh เก็บ range+view ใน query param (ไม่รีเซ็ต ไม่เด้งกลับ). custom ที่ from > to → ไม่ยิง query (error).
event = นับเหตุการณ์ที่เกิดในช่วงที่เลือก (เช่น PO ที่สร้างในเดือนนี้) · state = snapshot ปัจจุบันไม่ขึ้นกับช่วง (เช่น สต็อกใกล้หมดตอนนี้). backend คิดตามชนิดที่ระบุในตารางแต่ละ tile.

รูปแบบ endpoint (ต่อ tile มี 2 ตัว)

GET/api/dashboard/{dept}/summary?range=&from=&to=
ตัวเลข + caption ("ในช่วง"/"ตอนนี้") ของทุก tile ในแผนก + เวลา refresh ล่าสุด. Read {dept}
GET/api/dashboard/{dept}/tiles/{tileKey}?range=&from=&to=&page=&pageSize=
drill-down: list + pagination + คอลัมน์ตามตาราง + deepLink ต่อแถว. tile=0 → empty state "ไม่มีรายการ". เปลี่ยน range ระหว่าง drill เปิด → tile ชนิด event รีคำนวณ.
