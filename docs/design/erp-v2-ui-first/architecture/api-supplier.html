API Spec — Supplier + Price Matrix

เพิ่ม/แก้ supplier + ผูกวัตถุดิบรายตัวพร้อมราคารับซื้อ (matrix supplier×วัตถุดิบ) — ค้นวัตถุดิบตอนผูกได้. Active/Inactive. ไม่มีฟังก์ชันรับเข้าคลังในหน้านี้ (อยู่ที่ goods-receipt). ราคานี้ใช้คิด "ราคาทุน BOM = สูงสุดของ supplier ที่ active".

อ่าน / ค้นหา

GET/api/suppliers
รายการ + pagination + filter active/inactive. Read supplier
GET/api/suppliers/{id}
รายละเอียด + วัตถุดิบที่ผูก + ราคารับซื้อรายตัว (matrix) + lot_prefix.
GET/api/materials/search?q=
ค้นวัตถุดิบ (ใช้ตอนผูกเข้า supplier) — ซ่อนวัตถุดิบที่ถูกลบ (ADR-009).

สร้าง / แก้ + price matrix

POST/api/suppliers
สร้าง supplier + lot_prefix + ผูกวัตถุดิบ+ราคารับซื้อหลายรายการ. Create supplier
PATCH/api/suppliers/{id}
แก้ข้อมูล / เพิ่ม-ลบ-แก้ราคาใน matrix (audit ทุกช่อง). แก้ราคา → BOM ที่อ้างอาจขึ้น badge "ราคาทุนล้าสมัย".
POST/api/suppliers/{id}/toggle-active
Active↔Inactive · inactive แนะนำใช้แทนลบ · supplier inactive ไม่ถูกนับใน "ราคาทุน max active" ของ BOM.
DEL/api/suppliers/{id}
soft-delete (ห้ามซื้อใหม่, Lot เดิมใช้ได้). Delete supplier · comment บังคับ

error

codeเมื่อไร
SUPPLIER_MATERIAL_DUPผูกวัตถุดิบซ้ำใน matrix
MATERIAL_REFERENCE_DELETEDผูกวัตถุดิบที่ถูกลบ
