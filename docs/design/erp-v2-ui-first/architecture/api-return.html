API Spec — Return (คืน supplier)

คืนวัตถุดิบที่เสียหาย/QC ขาเข้าไม่ผ่าน: ระบุ lot → auto แสดง supplier → แก้จำนวน → ตัด stock + comment บังคับ (adjust ที่ไม่มี PO). ตัด stock = แถว movement reason=return_out (trace เสมอ).

สร้าง / คืน

GET/api/lots/{id}
ดึงข้อมูล lot → auto แสดง supplier + ยอดคงเหลือ (ใช้ตอนทำใบคืน).
POST/api/returns
สร้างใบคืน (ร่าง): lines[{lot_id, qty, reason}]. Create stock
POST/api/returns/{id}/confirm
→คืนแล้ว · ตัด stock (movement return_out) · noti Stock (C12). Update stock · comment บังคับ
Lot QC ขาเข้าไม่ผ่าน (C18) → เสนอสร้างใบคืนนี้อัตโนมัติ (ดู api-qc). ใบคืน = void ไม่ลบ.
