สิ่งที่แก้ในรอบ 4 (ตอบ feedback ปอนด์ ร3 + PO §8 + คำตอบ Batch/rework ล่าสุด)

P0 (flow/พฤติกรรม) · P1 (จอที่ขาด) · P2 (คุณภาพ) + ครบ 4 ประเด็น 🔍PO

P0: หน้าผลิตตัด "QC ไม่ผ่าน" ออก (ตัดสิน QC ที่หน้า QC เท่านั้น) · QC ตัดสินราย line/แบตช์ (ไม่ผ่าน→ตีกลับเฉพาะแบตช์+feedback บังคับ) · Batch lifecycle เลข B-{PO}-{line}-{run} สร้างตอน "เริ่มผลิต" (1 line=1 แบตช์) + rework badge/feedback ราย line · เห็น PO↔แบตช์↔Lot · หน้า create จริงกดได้: เพิ่มลูกค้า/ผู้ติดต่อ/Supplier/PR/BOM/รอบจัดส่ง
P1: Goods Receipt multi-line (header + หลายบรรทัด, lot รายบรรทัด, อ้างหลาย PR, partial→dialog สร้าง PR) · Dashboard date filter + caption/tile (ในช่วง vs ตอนนี้, default เดือนนี้) · Shipping สร้างรอบ 2 ทาง + คนขับ/เบอร์/route/ประเภทรถ · Supplier layout เต็มจอ (เลิก panel แคบ) + เพิ่ม Supplier จริง · po-detail UI เปลี่ยนสถานะ PO + แบตช์/QC ราย line
P2: ตัด label ซ้ำทุกหน้า (เหลือ breadcrumb + heading เดียว/หน้า — แก้ "Dashboard 3 จุด") · dropdown search ได้ทุกจุด (supplier/วัตถุดิบ/ลูกค้า/PO/Sale/route) · test data สมจริง (follow-up=เคส rework, blacklist มีเหตุผล, PO/Batch/Lot โยงกันจริง)
คำตอบปอนด์ที่ฝังรอบนี้: เลข Batch = B-{PO}-{line}-{run} · QC ตีกลับเฉพาะ line · Dashboard filter เดือน/ปี/ช่วงอิสระ (default เดือนนี้) · GR รับไม่ครบ → dialog "สร้าง PR ใหม่สำหรับของที่ขาด" · BOM ไม่มี supplier active → block จนกรอกราคาทุน
