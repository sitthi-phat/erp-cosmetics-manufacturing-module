3) คำถามถึงปอนด์ (ตอบที่ Gate 1)

ยืนยัน 3 เรื่องนี้เพื่อให้ Engineer เริ่ม implement ได้ตรงตามที่ต้องการ

คำถาม 1 · โทนสี: เลือกธีมไหน?  (ก) Clean Clinical — teal ✅แนะนำ   (ข) Warm Cosmetic — rose   (ค) Corporate Blue   (ง) มีสีแบรนด์เฉพาะ (กรุณาระบุ hex)
คำถาม 2 · โลโก้: มีไฟล์โลโก้จริงสำหรับหัวเว็บ + หัวใบกำกับภาษีไหม?  (ก) มี ส่งไฟล์   (ข) ยังไม่มี ใช้ชื่อบริษัทเป็นตัวหนังสือไปก่อน
คำถาม 3 · ชื่อ/ข้อมูลบริษัทบนเอกสาร: ชื่อ-ที่อยู่-เลขผู้เสียภาษีที่ใช้จริงบนใบกำกับภาษี (ตอนนี้ใส่ตัวอย่าง "บริษัท คอสเมดิก แฟคทอรี จำกัด")?  (ก) ใช้ข้อมูลจริงตามที่จะส่งให้   (ข) ให้ Admin กรอกเองในหน้าตั้งค่า
