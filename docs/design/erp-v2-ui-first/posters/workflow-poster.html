04 ตัดสินรายรุ่น

🔬 ฝ่ายควบคุมคุณภาพ (QC)

ผู้ทำ: เจ้าหน้าที่ QC

ตรวจ Batch ทีละรุ่น (ดูสายผลิต Lot→Batch→PO ตาม GMP) แล้วตัดสิน ผ่าน หรือ ไม่ผ่าน (ถ้าไม่ผ่านต้องระบุเหตุผล)

Batch: QC ผ่านBatch: QC ไม่ผ่าน
ผ่าน QC ทุกสินค้าในใบ?
✓ ผ่านครบ
สินค้าทุกบรรทัดของ PO ผ่าน → PO เปลี่ยนเป็น "พร้อมจัดส่ง" ส่งต่อฝ่ายจัดส่ง (ไปตอนที่ 2)
✕ ไม่ผ่าน
ตีกลับเฉพาะสินค้าตัวที่เสีย → ฝ่ายผลิตกด "ผลิตซ้ำ" (ออก Batch รุ่นใหม่ …-2-2) แล้วส่งตรวจอีกครั้ง · สินค้าตัวอื่นเดินหน้าต่อไม่สะดุด
