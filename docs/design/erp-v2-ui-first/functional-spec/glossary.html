ศัพท์สำคัญ

PRDใบสั่งผลิต (Production Order) — 1 ใบ/1 line item · สร้างเมื่อฝ่ายผลิต "รับงาน"
Batchรุ่นการผลิตจริง 1 รอบ (run) ของ PRD · gen ตอน "เริ่มผลิต" · +run เมื่อ rework
Lotล็อตวัตถุดิบขาเข้า (ต่อ supplier/รอบรับ) — ต่างจาก Batch (รุ่นผลิต FG)
Negative stockผลิตล่วงหน้าก่อนของเข้า → ยอดติดลบชั่วคราว → GR บวกกลับ
